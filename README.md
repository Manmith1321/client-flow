# 🚀 Client Flow

A full-stack **Client & Project Management System** built using Node.js and Express.
This application helps manage clients, projects, and tasks efficiently through a structured REST API and simple frontend interface.

---

## 📌 Features

* 👤 Client Management (Add, update, delete clients)
* 📁 Project Tracking
* ✅ Task Management System
* 🔗 RESTful API Architecture
* 📊 Organized backend (MVC structure)

---

## 🛠️ Tech Stack

**Frontend:**

* HTML
* CSS
* JavaScript

**Backend:**

* Node.js
* Express.js

**Others:**

* REST APIs
* MVC Architecture

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/Manmith1321/client-flow.git
cd client-flow
```

Install dependencies:

```bash
npm install
```

Run the server:

```bash
node server.js
```

---

## 🌐 API Endpoints (Sample)

| Method | Endpoint     | Description     |
| ------ | ------------ | --------------- |
| GET    | /clients     | Get all clients |
| POST   | /clients     | Add new client  |
| PUT    | /clients/:id | Update client   |
| DELETE | /clients/:id | Delete client   |
